Fensterbeschreibung

Hier werden die Fenster von Videlibri und ihre Funktion beschrieben:
Das HauptfensterÜber dieses Fenster werden die Medien angezeigt und verlängert.
Es wird (abgesehen vom Autostartmodus) automatisch bei Programmstart geöffnet.
Der EinstellungsdialogHiermit werden alle Einstellungen von Videlibri festgelegt.
Der Dialog wird über das Hauptfenster geöffnet.
Der Neue Konto-DialogÜber diesen Dialog werden alle neuen Kontos registriert.
Er lässt sich über den Einstellungsdialog erreichen.
Das BibTex-ExportfensterHiermit kann man die Medien als BibTex-Datien speichern.
Das StatistikfensterHier wird eine Statistik der ausgeliehenen Medien angezeigt.
Das MediensuchefensterIn diesem Fenster kann man nach Medien suchen.
Das FehlerrückmeldungsfensterDieser Dialog dient zum Melden von auftretenden Fehlern.
